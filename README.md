# 🎂 Online Cake Ordering Application

## 📌 Introduction
The **Online Cake Ordering Application** is a web-based system that allows customers to browse, customize, and order cakes online بسهولة.

This platform provides a smooth and user-friendly experience where customers can:
- Search cakes
- Customize orders
- Place and track orders

On the other hand, the **Admin** manages products, orders, payments, and overall business operations.

---

## 🚀 Features

### 👨‍💼 Admin Panel

- Admin Login / Logout
- Manage Cake Categories (Add, Update, Delete)
- Add New Cakes (ID, Name, Price, Image, Description)
- Update/Delete Cake Details
- View & Manage Customer Orders (Accept/Reject)
- Verify Payment Vouchers
- Confirm Orders with Estimated Delivery Time
- Generate Payment Vouchers
- View Customized Cake Requests
- Provide Feedback on Custom Orders
- Manage Business Expenses
- Generate Financial Reports (Income, Profit & Loss)

---

### 👤 Customer Panel

- User Registration & Login
- Profile Management
- Browse & View Cakes
- Search Cakes by Category
- Add to Cart (Multiple Cakes)
- Update Cart (Auto Price Update)
- Place Orders with Delivery Details
- Download Order Voucher
- Upload Payment Proof (Voucher)
- Track Order Status:
  - Processing
  - Confirmed
  - Dispatched
  - Delivered
- Track Estimated Delivery Time
- Request Custom Cakes
- View Admin Feedback on Custom Requests

---

## 🛠️ Technologies Used

- 💻 Frontend: HTML, CSS, Bootstrap
- 🧠 Backend: PHP (Procedural)
- 🗄️ Database: MySQL
- ⚙️ Server: XAMPP

---

## 📂 System Modules

### 1. Authentication Module
- Login / Registration system
- Role-based access (Admin / Customer)

### 2. Cake Management Module
- Category & Cake handling

### 3. Order Management Module
- Cart system
- Order placement & tracking

### 4. Payment Module
- Voucher generation
- Payment proof upload & verification

### 5. Custom Cake Module
- Request submission
- Admin feedback system

### 6. Reporting Module
- Expense tracking
- Financial reports

---

## 📊 Workflow

1. Customer registers & logs in  
2. Browse/search cakes  
3. Add items to cart  
4. Place order & get voucher  
5. Upload payment proof  
6. Admin verifies & confirms order  
7. Customer tracks order
